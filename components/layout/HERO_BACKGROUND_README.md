# Hero Background Configuration

## Описание

Компонент Hero теперь поддерживает фоновые изображения и видео с настраиваемым белым оверлеем и паралакс эффектом. Секция занимает 100% высоты экрана (`min-h-screen`) и контент вертикально центрирован для создания красивого полноэкранного эффекта с ощущением парения в облаках.

## Настройка

В файле [`components/layout/Hero.tsx`](Hero.tsx) найдите константу `BACKGROUND_CONFIG`:

```typescript
const BACKGROUND_CONFIG = {
  // Фоновое изображение
  image: '/images/hero-background.png',
  
  // Фоновое видео
  video: undefined as string | undefined,
  // video: '/videos/hero-background.mp4',
  
  // Прозрачность белого оверлея (0-1)
  overlayOpacity: 0.5,
};
```

## Использование фонового изображения

1. Поместите изображение в папку `public/images/`
2. Укажите путь к изображению в свойстве `image`:
   ```typescript
   image: '/images/hero-background.png',
   ```
3. Убедитесь, что `video` установлен в `undefined`:
   ```typescript
   video: undefined as string | undefined,
   ```

## Использование фонового видео

1. Поместите видео файл в папку `public/videos/`
2. Закомментируйте свойство `image`:
   ```typescript
   // image: '/images/hero-background.png',
   image: undefined,
   ```
3. Раскомментируйте и настройте свойство `video`:
   ```typescript
   video: '/videos/hero-background.mp4',
   ```

## Настройка прозрачности оверлея

Измените значение `overlayOpacity` (от 0 до 1):

- `0` - полностью прозрачный (фон виден полностью)
- `0.5` - полупрозрачный (по умолчанию)
- `1` - полностью непрозрачный (фон не виден)

```typescript
overlayOpacity: 0.5, // 50% прозрачности
```

## Настройка паралакс эффекта

### Включение/отключение паралакса

```typescript
parallax: {
  enabled: true, // true - включен, false - выключен
  speed: 0.5,
}
```

### Настройка скорости паралакса

Параметр `speed` определяет, насколько быстро фон движется при скролле:

- `0.1` - очень медленное движение (едва заметный эффект)
- `0.3` - медленное движение (мягкий эффект)
- `0.5` - среднее движение (по умолчанию, создает ощущение парения)
- `0.7` - быстрое движение (выраженный эффект)
- `1.0` - очень быстрое движение (максимальный эффект)

```typescript
parallax: {
  enabled: true,
  speed: 0.5, // Рекомендуемое значение для эффекта "парения в облаках"
}
```

### Отключение паралакса

Если паралакс эффект не нужен:

```typescript
parallax: {
  enabled: false,
  speed: 0.5,
}
```

## Рекомендации

- **Размер изображения**: рекомендуется использовать изображения шириной 1920px и выше для поддержки больших экранов
- **Формат видео**: MP4 (H.264) для лучшей совместимости с браузерами
- **Размер видео**: оптимизируйте видео для веба, рекомендуемый размер файла до 5-10 МБ
- **Разрешение видео**: 1920x1080 или 1280x720 для баланса качества и производительности
- **Контраст**: используйте оверлей для улучшения читаемости текста поверх фона
- **Полноэкранный режим**: секция Hero занимает 100% высоты экрана (`min-h-screen`) с вертикально центрированным контентом
- **Паралакс эффект**: используйте скорость 0.5 для создания естественного ощущения парения в облаках
- **Производительность**: паралакс оптимизирован с `will-change-transform` и пассивными слушателями событий
- **Видео с паралаксом**: видео масштабируется на 110% (`scale-110`) для обеспечения плавного эффекта без черных полос

## Темная тема

Оверлей автоматически адаптируется к темной теме:
- В светлой теме: белый оверлей
- В темной теме: темно-серый оверлей (`bg-neutral-900`)

## Технические детали

### Паралакс эффект

Паралакс реализован с помощью:
- React hooks (`useState`, `useEffect`) для отслеживания позиции скролла
- CSS transform для плавной анимации
- `will-change-transform` для оптимизации производительности
- Пассивные слушатели событий (`{ passive: true }`) для улучшения производительности прокрутки
- `scale-110` для видео, чтобы избежать появления черных полос при паралакс эффекте

### Клиентский компонент

Компонент помечен как `'use client'`, так как использует браузерные API (`window.scrollY`) и React hooks для интерактивности.
